usage: .log [level]

This command turns on / off generic logging.  This feature is mainly for
developers.

level:
  off
  severe
  warning
  info
  config
  fine
  finer
  finest
  all

Generic logging is only available to JDBC drivers that support JDBC 4.2,
which requires JDK 1.8+.  Most drivers have their own specific ways of
logging.
